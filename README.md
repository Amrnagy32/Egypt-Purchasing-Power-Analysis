Egypt Purchasing Power Analysis

Objective:
Analyze whether purchasing power in Egypt increased or decreased between 2010–2024.

Data Sources:

World Bank – Inflation (CPI, annual %)

World Bank – Real GDP per capita

Methodology:

Cleaned & filtered Egypt data.

Converted data to long format.

Indexed Real GDP per capita (2010 = 100).

Compared trend over time.

Key Insight:
Real GDP per capita index reached ~127 in 2024
→ +27% compared to 2010 (long-term growth, despite shocks).

Tools Used:

Python

Pandas

Matplotlib / Seaborn

Jupyter Notebook
